Revolut JWT playground

The aim of this tool is to help you test your JWT connection to the Revolut for Business Open API. The main steps to get started with the Open API are:

JWT generation HOWTO:

  1. Generate a private/public key pair using
    > openssl genrsa -out privatekey.pem 1024
    > openssl req -new -x509 -key privatekey.pem -out publickey.cer -days 1825
  2. In your Revolut for Business Demo account, open API settings page
    upload the public key and set https://revolut-jwt-sandbox.glitch.me as the OAuth redirect URI.
  3. Click on the Authorize access; you'll be redirected to this page.

You'll then be instructed on further actions.
